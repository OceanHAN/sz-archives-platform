### 🗺️ 选型方案：Leaflet + 高德地图数据源

我们依然使用 **Leaflet** 框架，但将底图数据源配置为 **高德地图 (AutoNavi)** 的在线瓦片。

*   **满足需求**：
    1.  **中文地图**：直接加载高德的中文瓦片，地名、道路全中文。
    2.  **更新及时**：数据与高德官方同步，即时反映最新的城市建设。
    3.  **无缝集成**：相比直接使用高德 SDK，这种方式无需复杂的 API Key 配置，且能更好地支持我们在地图上叠加“1980年旧地图”图片，实现新旧对比。

### 📅 实施计划

#### 1. 安装依赖
在 `projects/frontend-mobile` 下安装：
*   `npm install leaflet`
*   `npm install @types/leaflet -D`

#### 2. 实现 MapView 组件 (`src/views/MapView.vue`)
*   **加载高德底图**：
    使用 `L.tileLayer` 加载高德瓦片地址：
    `https://webrd01.is.autonavi.com/appmaptile?lang=zh_cn&size=1&scale=1&style=8&x={x}&y={y}&z={z}`
*   **初始化视图**：默认定位到深圳市中心（市民中心附近）。
*   **功能还原**：
    *   添加“时间轴” UI（1980-2025）。
    *   实现交互：点击“1980”时，在地图上方叠加一张半透明的 1980 年深圳历史地图图片（如有），实现今昔对照。

#### 3. 配置路由与入口
*   在 `router/index.ts` 注册 `/map-view`。
*   在首页 `HomeView.vue` 绑定点击事件。
